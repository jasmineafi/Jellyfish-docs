# Login ke Sistem 
Halaman ini menjelaskan cara masuk ke aplikasi Jellyfish.

## Mengakses Halaman Login
1. Buka browser Anda (disarankan menggunakan Google Chrome atau Mozilla Firefox ver terbaru)
2. Masukkan alamat URL aplikasi JellyFish yang diberikan oleh administrator

## Proses Login
## Langkah-langkah login
1. **Masukkan Username** - ketik username atau email yang telah didaftarkan
2. **Masukkan Password** - ketik password akun Anda
3. **Klik Tombol Login** - Tekan tombol "Masuk" atau "Login" untuk mrlanjutkan

## Tips Keamanan
* Jangan bagikan password Anda kepada siapapun
* Gunakan pawwsord yang kuat (kombinasi huruf, angka, dan simbol)
* Logout setelah selesai menggunakan aplikasi, terutama di komputer bersama

## Lupa Password
Jika Anda lupa Password:
1. Klik link "Lupa Password" di halaman login
2. Masukkan email yang terdaftar
3. Cek Inbox email Anda untuk lik reset password
4. Ikuti intruksi di email untuk membuat password baru

## Masalah login
Jika mengalami masalah saat login:
| Masalah | Solusi | 
|---------|---------|
| Password Salah  | Pastikan caps lock tidak aktif, coba reset password | 
| Akun terkunci | Hubungi administrator untuk membuka kunci akun | 
| Halaman tidak muncul| Periksa koneksi internet, coba refresh halaman | 

## Langkah Selanjutnya
Setelah berhasil login, Anda akan diarahkan ke Dashboard. Pelajari [Navigasi Aplikasi](/guide/navigasi) untuk memahami cara menggunakan aplikasi.
