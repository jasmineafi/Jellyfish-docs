## Menu Collapse
* Klik ikon hamburger (=) untuk menyembunyikan/menampilkan sidebar
* Mode collapsed menampilkan hanya ikon menu
* Hover pada ikon untuk melihat submenu

## Navigasi Tab
Jellyfish menggunakan sistem tab untuk memudahkan multitasking:

## Membuka Tab Baru
* Klik menu akan membuka tab baru
* Klik kanan untuk opsi tambahan (buka di tab baru, refresh, dll)

## Mengelola Tab
| Aksi | Cara | 
|---------|---------|
| Tutup tab  | Klik ikon X pada tab |
| Tutup tab lain | Klik kanan > "Tutup Lainnya"| 
| Refresh Tab | Klik kanan > "Refresh"| 
| Pin tab | Klik kanan > "Pin Tab"| 

## Breadcrumb
Breadcrumd di bagian atas konten menunjukkan lokasi Anda saat ini:
```text
Dashboard > Proyek > Detail Proyek > Edit
```

Klik pada bagian breadcrumb untuk navigasi cepat ke level sebelumnya.

## Pencarian Global
### Menggunakan Pencarian 
1. Klik ikon pencarian (ğŸ”) di header atau tekan <kbd>Ctrl + K</kbd> untuk mencari.
2. Ketik kata kunci yang ingin dicari
3. Hasil pencarian akan muncul secara real-time
4. Klik hasil untuk langsung menuju halaman terkait

### Jenis Pencarian
* **Proyek** - Cari berdasarkan nama atau kode proyek
* **Pelanggan** - Cari berdasarkan nama atau kontak
* **Menu** - Cari halaman atau fitur

## Shortcut Keyboard
Gunakan shortcut untuk navigasi lebih cepat
| Shortcut | Fungi | 
|---------|---------|
| <kbd>Ctrl + K</kbd> | Buka pencarian global |
| <kbd>Ctrl + /</kbd> | Tampilan daftar shortcut | 
| <kbd>Esc</kbd>| Tutup modal/popup| 

## Notifikasi
### Jenis Notifikasi
* **Info** - Pemberitahuan umum
* **Sucsess** - Aksi berhasil dilakukan
* **Warning** - Peinngatan yang perlu diperhatikan
* **Errorr** - Kesalahan yang perlu di tangani

## Panel Notifikasi
1. Klik lonceng (ğŸ””) di header
2. Lihat daftar notifikasi terbaru
3. Klik notifikasi untuk melihat detail
4. Klik "Tandai Semua Dibaca" untuk menghapus badge

## Tips Navigasi
::: tip Tip Cepat
Gunakan pencarian global untuk navigasi tercepat ke halaman manapun.
:::

::: Info Catatan
Tab yang sudah dibuka akan tersimpan saat Anda logout dan login kembali.
:::
